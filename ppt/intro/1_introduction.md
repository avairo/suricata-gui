# 1. INTRODUCTION

## Overview
SENTINEL is a modern, AI-enhanced Security Operations Center (SOC) dashboard designed to visualize, analyze, and manage network security alerts generated by the Suricata Intrusion Detection and Prevention System (IDS/IPS). It transforms raw network logs into an intuitive, interactive graphical interface, complete with geographical IP mapping and an AI-powered chatbot assistant for actionable threat intelligence.

## Problem Statement
Network security analysts often face alert fatigue when dealing with the high volume of raw log data produced by systems like Suricata. Traditional text-based logs are complex, lack immediate context, and require significant time and expertise to decode. This makes it difficult to quickly prioritize critical threats and understand their implications, delaying incident response times.

## Scope of the System
The project encompasses the development of a full-stack web application featuring:
- A backend system that parses and manages Suricata alert logs.
- A frontend interactive dashboard with real-time visualizations and geographical threat mapping.
- Integration with the AI API to provide dynamically generated, easy-to-understand explanations for specific network threats.
- An interactive AI chatbot tailored for security analysis and assistance.

## Objective
The primary objective of this project is to streamline network security monitoring by providing a unified, user-friendly interface. Specifically, it aims to:
- Simplify the visualization and monitoring of Suricata network alerts.
- Accelerate incident response by using AI to explain complex threats in natural language.
- Provide visual and interactive tools, like an IP map and a security chatbot, to assist analysts in threat assessment and decision-making.
